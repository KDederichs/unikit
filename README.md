# kuikit

> Cross-Plattform UI kit based on styled-components

[![NPM](https://img.shields.io/npm/v/kuikit.svg)](https://www.npmjs.com/package/kuikit) [![JavaScript Style Guide](https://img.shields.io/badge/code_style-standard-brightgreen.svg)](https://standardjs.com)

## Install

```bash
npm install --save kuikit
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'kuikit'

class Example extends Component {
  render () {
    return (
      <MyComponent />
    )
  }
}
```
